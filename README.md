# Retail Order Analytics Pipeline – Alteryx & Power BI

## Overview
This project demonstrates an end-to-end data analytics workflow using Alteryx and Power BI.

The pipeline performs data wrangling, cleansing, transformation, and aggregation to generate structured datasets for analytics dashboards.

## Tools Used
Alteryx Designer
Power BI
Excel

## Workflow
Raw Dataset → Data Cleansing → Feature Engineering → Aggregation → Dashboard

## Key Analytics
Orders by Region
Orders by Category
Customer Segmentation
Shipping Time Analysis
